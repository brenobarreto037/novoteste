# Pong em Python

## Descrição

Este é um projeto simples que implementa o clássico jogo **Pong** em Python usando a biblioteca **Pygame**. O jogo permite que dois jogadores controlem as raquetes para rebater a bola e marcar pontos. A pontuação final é salva em um arquivo de texto.

## Funcionalidades

- Jogo de Pong para dois jogadores.
- Controle das raquetes com as teclas **W/S** para o jogador 1 e **Setas para cima/baixo** para o jogador 2.
- A pontuação é exibida durante o jogo.
- As pontuações das partidas são salvas em um arquivo de texto chamado `pontuacoes.txt`.

## Pré-requisitos

Antes de rodar o projeto, você precisa ter o Python 3.x instalado e a biblioteca **Pygame** configurada. Se você ainda não tem o Python, você pode baixá-lo [aqui](https://www.python.org/downloads/).
